Today's Astronomy Picture of the Day

One URL. Today's picture.

nasapicture.com is a proxy for NASA's Astronomy Picture of the Day. It gives you a stable URL that always returns today's APOD image — no API key, no JavaScript, no configuration.

Drop it into an <img> tag, use it as a wallpaper source, or fetch the JSON metadata. Video days are skipped automatically — you always get an image.

// That's it. That's the whole integration.

<img src="https://api.nasapicture.com/">

Endpoints

All routes are on api.nasapicture.com

/ get

Today's APOD in high resolution. Falls back to standard if HD isn't available.

/sd get

Standard resolution image. Lighter payload, same picture.

/info get

JSON metadata — title, date, explanation, image URLs, copyright.

/about get

Plain-text description with ASCII art. Good for terminals.

How it works

A Cloudflare Worker fetches today's APOD from NASA's API, then streams the image bytes directly to you. The response is cached at the edge — subsequent requests in the same hour are served instantly from Cloudflare's network.

If today's APOD is a video (it happens), the worker walks backwards up to 7 days to find the most recent image. You never get an error for something out of your control.
